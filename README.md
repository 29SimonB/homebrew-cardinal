# 29simonb Cardinal

## How do I install these formulae?

`brew install 29simonb/cardinal/<formula>`

Or `brew tap 29simonb/cardinal` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "29simonb/cardinal"
brew "<formula>"
```
